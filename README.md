# THE JEFFREY EPSTEIN DIRECTORY - Information Page

A standalone HTML page that displays information about FraudLab.live and the Jeffrey Epstein Directory application.

## Installation

Simply download or clone this repository and open `index.html` in any web browser. No server or additional dependencies required.

### Quick Start

1. Download or clone this repository
2. Open `index.html` in your web browser
3. The page will display information about FraudLab.live and provide a link to access the resources
